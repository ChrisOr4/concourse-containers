# What is inside ?

- `cf` - cloud foundry client
- `autopilot cli plugin` - cf plugin for hands-off, zero downtime application deploys: https://github.com/contraband/autopilot 
- `antifreeze cli plugin` - CF CLI plugin to detect if an app has unexpected ENV vars or services bound which are missing from the manifest: https://github.com/odlp/antifreeze
- `uaac` - CloudFoundry UAA Command Line Client
